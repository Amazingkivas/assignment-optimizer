# Prompt для генерации example JSON

Сгенерируй валидный JSON-файл для веб-приложения решения задачи о назначениях с ограничениями (метод Удзавы).

## Требования к структуре
- JSON должен содержать поля:
  - `n` (целое, >= 1)
  - `k` (целое, >= 1)
  - `c` — квадратная матрица `n x n` с числовыми значениями
  - `d` — массив из `k` квадратных матриц `n x n`
  - `b` — массив длины `k` с числовыми ограничениями
  - `lambda0` — массив длины `k` с начальными неотрицательными множителями
  - `max_iter` (целое, >= 1)
  - `eps` (число > 0)
  - `step_mode` (`"harmonic"` или `"constant"`)
  - `step_const` (число)

## Ограничения качества данных
- Используй только числовые значения (без `null`, без строк в числовых полях).
- Размерности должны совпадать строго:
  - `c.length === n`
  - каждая строка `c` длины `n`
  - `d.length === k`
  - каждая матрица в `d` размера `n x n`
  - `b.length === k`
  - `lambda0.length === k`
- Значения в `lambda0` должны быть >= 0.

## Цель
Сделай пример реалистичным и удобным для проверки интерфейса: `n` от 5 до 8, `k` от 1 до 3, значения матриц в разумном диапазоне (например, 0..30).

Выведи **только JSON** без пояснений.
